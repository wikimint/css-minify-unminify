# Contributing Guidelines

Thank you for your interest in improving this project.

## 💡 How You Can Contribute
- Suggest new features
- Report bugs or issues
- Improve documentation
- Share performance ideas

## 📌 Contribution Rules
- Keep suggestions clear and relevant
- Focus on CSS optimization improvements
- Avoid submitting unrelated content

## 🛠 Process
1. Open an issue
2. Describe your suggestion clearly
3. Wait for review and discussion

## ⚠️ Note
This is a proprietary project. Direct code contributions are limited, but ideas and feedback are welcome.

## 🙌 Support
If you find this tool useful, consider supporting the project.
