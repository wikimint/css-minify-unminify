# Usage Guide

## How to Minify CSS

1. Paste your CSS code into the input area
2. Click the "Minify" button
3. Copy the optimized output

## How to Unminify CSS

1. Paste compressed CSS code
2. Click the "Unminify" button
3. Get readable formatted CSS

## Tips

- Use minified CSS in production
- Use unminified CSS for editing and debugging
- Always test styles after optimization

## Best Practice

Combine CSS minification with:
- HTML minification
- JavaScript optimization

This improves overall website performance.
